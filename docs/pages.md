# 대시보드/스크립트 지도

## 공통
- 모든 페이지는 `data.load_*` 로더 사용을 전제로 함(2025년 중심). 팀/파트 매핑은 `config.py` 활용.
- `_part_view_base.render_part()`를 쓰는 페이지는 Won/생성 기준 리텐션·신규/포맷별 요약·상세 블록을 동일 UX로 노출.

## 주요 페이지
- `pages/00_2026 P&L Projection.py`
  - 2025 Won 데이터를 기반으로 2026년 매출/OP 시뮬레이션(Backlog, 삼성 월 계획, 온라인 리텐션+365, 출강 Upsell, 신규 Gap 채움).
  - 리드타임·교육기간 Median 룩업(상위채널×기업규모×S-tier → 상위채널×기업규모 → 상위채널 → 기본값) 및 Rule-of-50, 고정비(제작/마케팅/인건비/임대/기타) 반영.
  - 업로드 파일로 입력 교체 가능. TODO/검증 항목은 `TODO.md`, `context-bridge.md` 참고.
- 체결액/체결률 추이
  - `pages/01_2024~2025_체결액_추이.py`: Won 기준 연/분기/월별 체결액(억) 스택 차트, 포맷 필터.

  - `pages/13_사업부_온라인_체결률(리텐션).py`, `pages/14_사업부_온라인_체결률(신규).py`: 온라인 포맷만 필터, 리텐션/신규 체결률/예정액(미최초 입과 필터 포함).
